Arke Scholars — Question Paper Author Guide

How to write and format question papers using the Arke template so they import correctly — including equations, images, and match-the-following questions.

1. What are Paragraph Styles?

In Microsoft Word, every paragraph carries a Style — a named tag that tells the Arke system what each paragraph is (question number, stem text, option, answer, etc.), not just how it looks. Applying a style is one click and makes imports accurate every time.

If you skip styles, the parser falls back to guessing from formatting — which works for simple papers but is less reliable with equations and complex layouts. Always use styles.

2. The Six Arke Styles

Q-Number
Bold question number. One per question. Apply to the number line only.
1.
Q-Stem
The question text. Can span multiple paragraphs. Equations live here.
A particle with mass $m$ moves with velocity $v$. Find its KE.
Q-Option
One answer option per paragraph. Label with (1) (2) (3) (4).
(1) $\frac{1}{2}mv^2$    (2) $mv^2$    (3) $2mv^2$    (4) $mv$
Q-Answer
Correct answer. Must start with "Answer:". One per question.
Answer: (1)
Q-Solution
Full solution / explanation. Can be multiple paragraphs.
$KE = \frac{1}{2}mv^2$ by the work-energy theorem.
Q-Topic
Optional. Placed before Q-Number. Format: "Topic: <name>"
Topic: Kinematics

3. Full Question Examples

Single / Multiple Correct (SCQ / MCQ)

Q-Topic
Topic: Oscillations
Q-Number
1.
Q-Stem
In damped SHM the displacement is $x = a_0 e^{-bt/2m}\sin(\omega t+\alpha)$. Which statements are correct?
Q-Option
(1) Dimensions of $b/m$ are the same as $T^{-1}$
Q-Option
(2) Dimensions of $a_0\omega$ are $[LT^{-1}]$
Q-Option
(3) Dimensions of $1/b$ are $[MT^{-1}]^{-1}$
Q-Option
(4) Amplitude decays as $e^{-bt/2m}$
Q-Answer
Answer: (1), (2), (4)
Q-Solution
From $bt/2m$ being dimensionless: $[b]=[MT^{-1}]$, so $[b/m]=[T^{-1}]$ ✓. $[a_0\omega]=[LT^{-1}]$ ✓. $[1/b]=[M^{-1}T]\ne[MT^{-1}]^{-1}$, so (3) is wrong.

Integer Answer Type

Q-Topic
Topic: Kinematics
Q-Number
2.
Q-Stem
A particle starts from rest with acceleration $a=2t$ m/s². The distance (in m) covered in the first 3 s is:
Q-Answer
Answer: 9
Q-Solution
$v=\int 2t\,dt=t^2$. Distance $s=\int_0^3 t^2\,dt=[t^3/3]_0^3=9$ m.

Match the Following

After the Q-Stem paragraph, insert a two-column Word table. The header row must say Column A and Column B. Each data row uses the format (A) item in the left cell and (P) item in the right cell.

Q-Topic
Topic: Thermodynamics
Q-Number
3.
Q-Stem
Match each quantity in Column A with its SI unit in Column B.
Table
Column AColumn B
(A) Energy(P) Pascal (Pa)
(B) Power(Q) Joule (J)
(C) Pressure(R) Coulomb (C)
(D) Electric charge(S) Watt (W)
Q-Answer
Answer: A-Q, B-S, C-P, D-R
Q-Solution
Energy → Joule; Power → Watt; Pressure → Pascal; Charge → Coulomb.
Match table rules: The header row must contain the words "Column A" and "Column B" (or "Column I" / "Column II"). Keys in Column A use letters A, B, C, D and keys in Column B use letters P, Q, R, S. The answer format is Answer: A-Q, B-S, C-P, D-R using hyphens or arrows.

4. Step-by-Step: Creating a Question Paper

Download the template Admin panel → Upload Questions → click "Download Template". Opens arke-question-template.docx.
Open and save as your paper Double-click the downloaded file. Immediately do File → Save As → YourPaper.docx. The template itself stays unmodified.
Show the Styles panel Home tab → click the small arrow at the bottom-right of the Styles group. The 6 Arke styles appear in the panel on the right.
Type and tag each question
  1. Optionally type Topic: Waves → select → click Q-Topic.
  2. Type 1. → select → click Q-Number.
  3. Type the question text → select → click Q-Stem.
  4. For options: type (1) option text on its own line → Q-Option. Repeat for (2)(3)(4).
  5. For match-the-following: instead of option lines, insert a table (see Section 6).
  6. Type Answer: (2) → select → click Q-Answer.
  7. Type the solution → select → click Q-Solution.
Save as .docx and upload File → Save As → format Word Document (.docx). Upload on the Arke Admin → Upload Questions page.

5. Speed Tip — Keyboard Shortcuts for Styles

Right-click any style in the panel → Modify → Format → Shortcut key. Suggested bindings:

  • Alt+1 → Q-Number
  • Alt+2 → Q-Stem
  • Alt+3 → Q-Option
  • Alt+4 → Q-Answer
  • Alt+5 → Q-Solution
  • Alt+6 → Q-Topic

With shortcuts you can tag an entire question in seconds without touching the mouse.

6. Equations

Two methods are supported. Both import correctly — use whichever is faster for you.

Method 1 — Type LaTeX directly

Wrap inline math in $...$ and display math in $$...$$. Type it as plain text anywhere inside a Q-Stem, Q-Option, or Q-Solution paragraph.

Stem:  $x = a_0 e^{-bt/2m}\sin(\omega t)$

Option:  (1) $\frac{1}{2}mv^2$

Display:  $$\int_0^\infty e^{-x^2}\,dx = \frac{\sqrt{\pi}}{2}$$

Fastest approach. No special Word setup needed. Renders with KaTeX on the Arke platform.

Method 2 — Word Equation Editor

Press Alt+= to insert a Word equation object. Type LaTeX inside the box and press Space to render it. The Arke parser extracts the LaTeX automatically.

1. Click inside a Q-Stem paragraph.
2. Press Alt + = (inserts equation box).
3. Type: \frac{bt}{2m}
4. Press Space — Word renders it.
5. Click outside to return to text.

Better for complex multi-line expressions. Requires no additional setup in recent Word versions.

Tip — LaTeX input mode in Word: If pressing Space doesn't render the equation, enable Math AutoCorrect: File → Options → Proofing → AutoCorrect → Math AutoCorrect → check "Use Math AutoCorrect rules outside of math regions".
Avoid MathType for new papers. MathType OLE objects are imported as images (the EMF preview), which cannot be searched or edited on the platform. Use Word's built-in editor (Alt+=) or plain $...$ LaTeX instead.

7. Match the Following — Detailed Instructions

Write the stem Type the introductory text (e.g. "Match the following") and apply Q-Stem.
Insert a 2-column table Place your cursor on the line immediately after the stem. Go to Insert → Table → 2 columns, as many rows as you need (1 header + N data rows).
Fill the header row Type Column A in the left cell and Column B in the right cell. This is required — the parser detects match tables by the header text.
Fill data rows Each row: left cell = (A) item text, right cell = (P) item text. Use A/B/C/D for Column A and P/Q/R/S for Column B. You can include $...$ LaTeX inside cells.
Write the answer Apply Q-Answer to a line like: Answer: A-Q, B-S, C-P, D-R

8. Images and Diagrams

Images are supported in the question stem, inside options, and in solutions.

  • Stem image: Insert → Pictures at any point inside or after Q-Stem paragraphs.
  • Option image: After typing the option text paragraph, insert the image as a new paragraph immediately below it (still before the next option). The parser pairs the image with the preceding option automatically.
  • Solution image: Insert anywhere inside a Q-Solution paragraph or as a separate paragraph within the solution block.
Image formats: PNG, JPEG, GIF, SVG, WMF, EMF are all supported. Avoid copy-pasting from PDF viewers — use proper screenshots or exported vector files. Do not use HD Photo (.wdp) format — it is not supported by browsers.

9. Answer Line Formats

The answer line must begin with Answer: (capital A, colon). The format after the colon depends on question type:

  • Single correct (SCQ): Answer: (3)
  • Multiple correct (MCQ): Answer: (1), (2), (4)  or  Answer: (1) and (3)
  • Integer: Answer: 42  or  Answer: -6  or  Answer: 3.5
  • Match the following: Answer: A-Q, B-S, C-P, D-R

10. Common Mistakes

Do not leave question parts in Normal style. Without Arke styles, the system falls back to heuristic detection which is less reliable. Always apply a style to every paragraph.
Answer line must start with "Answer:". Don't write "Ans:", "A:", "Correct:", or leave it blank.
Match table header must say "Column A" and "Column B". Other names like "List I / List II" or "Item / Match" will not be detected as a match table.
Do not split a paper across multiple files. One upload = one paper. All questions must be in a single .docx file.
Option labels must be (1) (2) (3) (4). Using A/B/C/D as option labels for MCQ/SCQ questions will confuse the answer parser. Reserve A/B/C/D for match-the-following Column A.

11. Frequently Asked Questions

I have an existing paper — do I have to retype everything?

No. Open your existing paper and the template side by side. Copy paragraphs from your paper into the template document, then apply the Arke styles. You can also open the Styles pane in your existing doc, click "Manage Styles" → Import/Export, and import the Arke styles from the template file directly.

Can I mix $...$ LaTeX and Word equation objects in the same paper?

Yes. Both formats are parsed independently and rendered the same way on the platform. Use whichever is convenient per question.

What if the system shows "needs review" for some questions?

The admin review page flags questions where the parser wasn't fully confident — usually a missing answer, an unrecognised option format, or a question with no text stem. Click each flagged card to confirm or correct the stem, options, and answer before saving.

Can each option have its own image?

Yes. Type the option text paragraph (e.g. (1) See diagram with Q-Option style), then on the very next line insert the image (Insert → Pictures). The parser automatically assigns the image to option (1). Repeat for each option that has an image.

What if an option is entirely an image with no text?

Type just (1) as the option text, apply Q-Option, then insert the image on the next line. The label is still needed so the parser knows which option slot the image belongs to.

My match table has more than 4 rows — is that OK?

Yes. The table can have any number of data rows. The parser reads all rows below the header. Just make sure each left cell starts with (A), (B), etc. and each right cell with (P), (Q), etc.

Can I use LaTeX inside match table cells?

Yes. Type (A) $\frac{1}{2}mv^2$ directly in the cell. The system parses $...$ expressions inside table cells the same as in stem or option paragraphs.

Q-Solution is optional?

Yes. If you skip Q-Solution, the question imports without a solution. You can add solutions later from the admin question editor on the platform.
